欢迎来到 Hugo NexT 文档站点！ 它是从 Theme NexT 移植过来的为 Hugo 打造的高品质优雅主题，保持简单易用的特性和强大的功能。

欢迎加入 Hugo NexT 组织！

发表于： 2022年06月01日 更新于： 2022年07月03日 分类于： 发布

使用反馈

  • GitHub Issues 提交错误报告 🐛

支持在Markdown文档中实现块信息强调标注

发表于： 2025年01月02日 更新于： 2025年01月02日 分类于： 示例文章/语法 浏览：

过去 HugoNexT 主题都是通过自定义短语来实现块信息的标注，近期发现 Hugo 从 v0.134.0 版本开始便是可以支持通过hook方式渲染Markdown文档中的Blockquote样式（需要自己实现），实现类似Github的警告风格样式。于是便在 HugoNexT 主题中添加7种不同风格供用户可选择使用，还可以自定义图标与颜色配置。
