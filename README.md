## Getting started

Install Hugo(**extended**) & `Hugo NexT` & Document Website Source on localhost:

```bash
git clone -b source https://github.com/hugo-next/hugo-next-docs
cd hugo-next-docs
git submodule update --init
```

Run server:

```bash
chmod a+x start.sh
sh start.scratch
```
