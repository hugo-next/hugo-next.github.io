Support Emoji emojis

Publish on: 2025/01/25 Classify at: Example/Grammar
Words: 686 Read:≈ 4mins Views: Comments:
Summary
Usage guide for emojis in Hugo and NexT.

Emojis can be enabled in various ways in the Hugo project.

emojify The method can be called directly in the template, or use [Inline Shortcodes]( https://gohugo.io/templates/shortcode-templates#inline -shortcodes).

To use emojis globally, you need to configure them on your website set enableEmoji to true,Then you can directly enter the emoji code in the article.

They start and end with a colon and contain the Emoji code:

1
2
3

Go camping! {:}tent:  I'll be back soon

How happy! {:}joy:

The output effect presented is as follows:

Let’s go camping ⛺ I’ll be back soon.
I’m so happy 😂

The following symbol list is a very useful reference for emoji code.
